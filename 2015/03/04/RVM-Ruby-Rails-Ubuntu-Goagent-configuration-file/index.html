RVM, Ruby, Rails, Ubuntu Goagent configuration

Posted on 2015-03-04 
Installing required packages: gawk, g++, libreadline6-dev, zlib1g-dev, libssl-dev, libyaml-dev,
libsqlite3-dev, sqlite3, autoconf, libgdbm-dev, libncurses5-dev, automake, libtool, bison,
libffi-dev

安装vmtools

cd /tmp/vmware-tools-distrib/
sudo ./vmware-install.pl

升级系统

sudo apt-get update
sudo apt-get upgrade

安装 RVM

gpg --keyserver hkp://keys.gnupg.net --recv-keys 409B6B1796C275462A1703113804BB82D39DC0E3
sudo apt-get install curl
curl -sSL https://get.rvm.io | bash -s stable
source ~/.rvm/scripts/rvm
rvm -v

source ~/.bashrc
source ~/.bash_profile

修改 RVM 的 Ruby 安装源到国内的 淘宝镜像服务器，这样能提高安装速度

sed -i -e 's/ftp\.ruby-lang\.org\/pub\/ruby/ruby\.taobao\.org\/mirrors\/ruby/g' ~/.rvm/config/
db

列出已知的ruby版本

rvm list known

用 RVM 安装 Ruby 环境

rvm install 1.9.3
rvm install 2.2.0

设置 Ruby 版本

rvm use 1.9.3

下面的命令将指定版本的 Ruby 设置为系统默认版本

rvm use 1.9.3 --default

查询已经安装的ruby

rvm list

检查版本

ruby -v
gem -v

卸载一个已安装版本

rvm remove 1.9.3

安装 Rails 环境

gem source -r https://rubygems.org/
gem source -a https://ruby.taobao.org
gem install rails -v 3.2.13
rails -v

sudo apt-get install vim

安装配置goagent

打开vpn登录

https://github.com/goagent/goagent
http://git.io/goa

cd software/goagent/goagent-goagent-437a921/local/ 编辑procy.int文件

APPID:


cd software/goagent/goagent-goagent-437a921/server/ 上传uploader.py

python uploader.py
APPID:
Email:

导入证书

sudo apt-get install python-vte
sudo python goagent-gtk.py

关于ubuntu开机启动goagent程序的方法

在系统工具中有一个“启动应用程序”工具，输入终端下的命令点击保存，重启即可。
python addto-startup.py

启动goagent

python proxy.py
