部署hexo

Posted on 2015-03-04

注意：教程针对Windows用户

  1. 安装Git
    下载 msysgit 并执行即可完成安装。
  2. 安装Node.js
    在 Windows 环境下安装 Node.js 下载安装文件(我选择的是.msi格式的安装包)并执行即可完成安装。(注意安装完后环境变量可能还没生效，git bash 提示找不到npm命令，可以注销一下)

  3. 安装hexo
    利用 npm 命令即可安装。
    npm install -g hexo

  4. 创建hexo文件夹
    安装完成后，在你选择的文件夹下，执行以下指令Hexo会自动在目标文件夹建立网站所需要的所有文件。
    hexo init

  5. 安装依赖包
    npm install
  6. 本地查看
    现在我们已经搭建起本地的hexo博客了，执行以下命令(在\hexo目录下)，然后到浏览器输入localhost:4000查看。
     hexo generate
 hexo server

至此，本地博客已经搭建起来了，只是本地哦，别人看不到的。

Generating SSH keys

ssh-keygen -t rsa -C "your_email@example.com"
ssh-agent -s
eval `ssh-agent`
ssh-add ~/.ssh/id_rsa
ssh-agent -s

Add your SSH key to your account!!!

ssh -T git@github.com
ssh -T git@coding.net

win7 and ubuntu

Posted on 2015-03-04

win7 ubuntu双系统安装方法

  1. 准备一个安装ubuntu的分区及Ubuntu的ISO镜像

  1. 将ubuntu系统镜像拷贝到C盘根下。把ubuntu系统镜像文件中casper目录下 initrd.lz vmlinuz （13.10中64位：vmlinuz.efi）拷贝出来放到C盘根下（13.10中64位的vmlinuz.efi改名为vmlinuz）。

  2. windows下运行EasyBCD中“添加新条目”->“NeoGrub”->“安装”

  3. 配置，安装完成后点击“配置”，输入如下内容：

    title Install Ubuntu
root (hd0,1)
kernel (hd0,1)/vmlinuz boot=casperiso-scan/filename=/ ubuntu-14.10-desktop-i386.iso ro 
quiet splashlocale=zh_CN.UTF-8
initrd (hd0,1)/initrd.lz

==注意配置中的待安装的ubuntu系统镜像文件名和指定C盘所在分区的位置，注意调节括号中后面一个数值。==
